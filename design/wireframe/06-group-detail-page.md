# 頁面 06：群組詳情頁（Group Detail Page）

## 頁面概述

進入群組後的主畫面，包含三個 Tab：總覽、債務清算、成員與動態。

---

### 功能按鈕

| 元素 | 功能 |
|------|------|
| 帳務卡片（點擊） | 展開帳務詳情（品項、金額、付款人、分帳對象、備注、圖片）|
| 帳務卡片（長按或滑動） | 顯示「編輯」/ 「刪除」選項 |
| ＋ 新增帳務（浮動按鈕） | 開啟新增帳務頁面 |
| ⋯ 更多（右上角） | 顯示選單：「分享邀請連結」/ 「編輯群組資料」/ 「離開群組」|

### 功能按鈕

| 元素 | 功能 |
|------|------|
| 全部標記完成 | 將所有未完成清算項目標記為已完成 |

| 已完成項目 ✅ | 可再次點擊取消完成狀態 |
| 進度條 | 視覺化清算完成比例 |

## 設計備注

- Tab 切換使用滑動手勢（左右滑動切換 Tab）
- 浮動 ＋ 按鈕固定在畫面右下角，不遮蓋內容
- 帳務卡片金額文字使用橙色強調
- 未綁定佔位成員頭像顯示灰色預設圖示 + 文字首字
