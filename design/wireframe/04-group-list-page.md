# 頁面 04：群組列表頁（Group List Page）

## 頁面概述

顯示使用者加入或建立的所有群組，並提供建立新群組的入口。

## 功能說明

### 頂部操作

| 元素 | 功能 |
|------|------|
| ＋ 新增按鈕 | 進入建立群組頁面 |
| 搜尋框 | 即時篩選群組名稱 |

### 群組卡片

| 元素 | 功能 |
|------|------|
| 點擊任意卡片 | 進入該群組詳情頁 |
| 長按卡片（可選） | 顯示快捷選單：「分享邀請連結」/ 「刪除群組」|

### 分組顯示邏輯

- **進行中**：尚有未結清帳款的群組
- **已結清**：所有帳款皆已結清的群組（顯示灰色調，視覺較低調）

---

## 設計備注

- 封面圖為 16:9 縮圖，無封面圖則顯示帶群組名稱首字的色塊
- 金額文字：欠別人 → 橙色，別人欠你 → 綠色，已結清 → 灰色打勾
- 搜尋框無結果時顯示空狀態插圖：「找不到符合的群組」
- 列表為空時顯示引導文字：「你還沒有任何群組，點擊 ＋ 建立第一個！」
