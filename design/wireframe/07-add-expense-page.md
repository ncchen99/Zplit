# 頁面 07：新增帳務頁（Add Expense Page）

## 頁面概述

讓使用者輸入一筆帳務的詳細資訊，支援多種分帳模式、圖片上傳與重複記帳設定。

## 功能說明

### 頂部操作列

| 元素 | 功能 |
|------|------|
| ✕ 取消 | 放棄並返回，彈出確認提示 |
| 儲存 | 驗證必填欄位後，寫入 Firestore |

### 付款人選擇

| 元素 | 功能 |
|------|------|
| 付款人下拉 | 顯示所有群組成員，點擊選擇付款人 |

### 分帳對象區

| 元素 | 功能 |
|------|------|
| 勾選框 | 點擊勾選/取消該成員參與分帳 |
| 全選 | 勾選所有成員 |
| 清除全部 | 取消所有勾選 |
| 金額 / 百分比輸入框 | Split by Amount / Percent 模式下出現 |

### 圖片上傳

| 元素 | 功能 |
|------|------|
| 📷 新增圖片 | 開啟選擇器（相機 / 相簿），上傳至 Cloudflare R2 |
| 圖片預覽 ✕ | 點擊移除該張圖片 |

---

## 設計備注

- 品項名稱與金額為必填，未填時儲存按鈕呈 disabled
- Simple Split 下，金額會即時計算並顯示每人分攤額
- 分帳模式切換時，已輸入的數值會清除並重新計算
- 頁面為長頁面，可向下捲動，「儲存」按鈕固定在頂部
