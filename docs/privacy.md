# 隱私政策 (Privacy Policy)

Zplit 非常重視您的隱私權。本政策說明我們如何收集、使用及保護您的個人資訊。

## 1. 資訊收集
- **帳號資訊**：當您使用 Google 登入時，我們會收集您的顯示名稱、電子郵件地址及大頭貼連結。
- **使用資料**：我們會收集您在服務中建立的群組名稱、帳務品項、金額、日期及備註。
- **圖片資料**：若您上傳收據或封面圖，這些圖片將儲存在我們的雲端空間。

## 2. 資訊使用
我們收集資訊的目的是為了：
- 提供與維持分帳服務。
- 讓群組成員之間能看到彼此的帳務記錄。
- 改善服務品質。

## 3. 資訊分享
我們不會將您的個人資訊出售或分享給第三方，除非：
- 經您同意。
- 為了提供服務（例如儲存在 Firebase 資料庫）。
- 法律要求。

## 4. 資料安全
我們使用 Firebase 等業界標準技術來保護您的資料安全。然而，網路傳輸並非百分之百安全，我們無法保證絕對的安全。

## 5. 您的權利
您可以隨時透過設定頁面編輯您的個人資料，或選擇刪除帳號以移除所有相關數據。

## 6. 政策修改
我們可能會不時更新本隱私政策。建議您定期查看本頁面以瞭解最新資訊。
