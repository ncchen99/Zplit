## 功能說明

### 頂部操作

| 元素 | 功能 |
|------|------|
| ← 返回 | 返回個人借貸總覽頁 |
| ⋯ 更多 | 顯示選單：「結清所有帳款」/ 「編輯對象名稱」/ 「刪除聯絡人」|

### 最佳分帳開關（Optimal Split）

| 元素 | 功能 |
|------|------|
| 開關切換 | 開啟時，系統掃描是否有共同群組，並整合計算最優清算方案 |
| 說明文字 | 顯示目前偵測到的共同群組（例如「找到共同群組：墾丁旅遊 2026」）|

### 借貸記錄卡片

| 元素 | 功能 |
|------|------|
| 點擊卡片 | 展開借貸詳情（備注、圖片、分帳細節）|
| 長按或右滑卡片 | 顯示「編輯」/ 「刪除」|
| ＋ 新增借貸（浮動按鈕） | 開啟新增帳務頁，預設付款人為自己、對象為心怡 |

### ⋯ 更多選單

| 選項 | 功能 |
|------|------|
| 結清所有帳款 | 新增一筆結清紀錄，淨額歸零 |
| 編輯對象名稱 | 修改這位聯絡人的顯示名稱 |
| 刪除聯絡人 | 刪除所有與此人的記錄（需二次確認）|

---

## 設計備注

- 淨額文字大而清晰：綠色 = 別人欠你，橙色 = 你欠別人
- Optimal Split 開關開啟後，若偵測到共同群組，頁面上方顯示黃色提示條：「已找到共同群組，建議透過群組清算功能統一結算」
- 若對方也有 Zplit 帳號且已綁定，頭像下方顯示「已連結」小標籤
- 結清記錄以灰色卡片顯示，視覺較低調
