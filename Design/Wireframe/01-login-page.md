# 頁面 01：登入頁（Login Page）

## 頁面概述

使用者進入 App 的第一個畫面。畫面設計乾淨、聚焦，目的是讓使用者快速完成登入。

## 功能說明

### 按鈕與互動

| 元素 | 功能 |
|------|------|
| Google 登入按鈕 | 觸發 Firebase Auth Google Provider 登入流程 |
| 匿名快速體驗按鈕 | 觸發 Cloudflare Turnstile 人機驗證流程 |
| 使用條款 / 隱私政策 | 開啟對應說明頁面（可為外部連結）|

### 匿名登入流程（按下後）

1. 頁面出現 Cloudflare Turnstile 驗證元件（滑塊或勾選框）
2. 驗證通過後，自動建立 Firebase 匿名帳號
3. 導向新使用者資料填寫頁（若為首次）或直接進入主頁

---

## 設計備注

- 背景使用品牌主色（lemonade 主題淺色 / dim 主題深色）
- Logo 置中，佔頁面上方 40% 空間
- 兩個按鈕大小一致，Google 按鈕為主要 CTA（顏色較深）
- 匿名按鈕為次要樣式（outline 或 ghost）
