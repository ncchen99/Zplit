# 頁面 03：首頁（Home Page）

## 頁面概述

登入後的主要入口頁面，顯示使用者目前所有群組與個人分帳的快速摘要，並提供快速新增入口。

## 功能說明

### 頂部導覽列

| 元素 | 功能 |
|------|------|
| Zplit Logo / 文字 | 點擊回到首頁頂部 |
| 頭像圖示 | 點擊進入個人設定頁 |
| 鈴鐺通知圖示 | 點擊展開通知列表（如：有人新增帳務） |

### 群組區塊

| 元素 | 功能 |
|------|------|
| 群組卡片 | 點擊進入該群組詳情頁 |
| 「全部」連結 | 進入完整群組列表頁 |

### 個人分帳區塊

| 元素 | 功能 |
|------|------|
| 聯絡人卡片 | 點擊進入與該對象的個人分帳詳情頁 |
| 「全部」連結 | 進入個人分帳總覽頁 |

### 底部導覽列（Bottom Nav）

| 圖示 | 功能 |
|------|------|
| 首頁 | 返回首頁 |
| 群組 | 進入群組列表頁 |
| ＋（中央大按鈕） | 開啟新增選單：「新增群組帳務」或「新增個人分帳」|
| 個人 | 進入個人分帳總覽頁 |
| 設定 | 進入個人設定頁 |

---

## 設計備注

- 摘要文字依實際狀況動態顯示（無待結清時顯示「目前沒有待結清帳款！」）
- 群組卡片顯示封面圖縮圖（無封面則顯示預設漸層色塊）
- 「你欠 $XXX」以橙色文字顯示，「別人欠你」以綠色顯示
- 首頁不顯示全部資料，最多顯示 3 筆群組 + 4 筆個人分帳，更多需點擊「全部」
