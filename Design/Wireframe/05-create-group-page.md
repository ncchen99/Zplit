# 頁面 05：建立群組頁（Create Group Page）

## 頁面概述

讓使用者設定新群組的名稱、封面圖，並預先新增成員。完成後立即產生邀請連結。

## 功能說明

### 頂部操作列

| 元素 | 功能 |
|------|------|
| ✕ 取消 | 放棄並返回上一頁，彈出確認提示 |
| 完成 | 等同於建立群組按鈕 |

### 封面圖上傳

| 元素 | 功能 |
|------|------|
| 封面圖區域（點擊） | 開啟選擇器：「拍攝照片」/ 「從相簿選取」/ 「取消」|
| 上傳中狀態 | 區域顯示 Loading 動畫 |
| 上傳完成 | 顯示預覽圖，右上角出現「重新上傳」小按鈕 |

### 成員搜尋與新增

| 元素 | 功能 |
|------|------|
| 搜尋框 | 輸入名字即時搜尋歷史聯絡人 |
| 搜尋結果列表 | 顯示現有聯絡人（按互動頻率排序），點擊即加入 |
| 「+ 新增『XXX』作為成員」 | 輸入新名字後即時顯示，點擊直接新增未綁定佔位成員 |
| 已新增成員頭像 ✕ | 點擊移除該成員（群主自己不可移除） |

### 建立群組按鈕

- 群組名稱為空時：按鈕呈 disabled 狀態
- 點擊後：顯示 Loading → 建立成功後自動跳轉至該群組詳情頁
- 建立完成後立即自動產生邀請連結（背景處理，不影響使用者體驗）

---

## 設計備注

- 自己（群主）預設已加入成員列表，頭像下方顯示皇冠圖示 👑
- 成員頭像列表以橫向排列，超過 5 人則顯示「+N 人」
- 群組名稱輸入框下方可加入字數計數（例如：0/30）
